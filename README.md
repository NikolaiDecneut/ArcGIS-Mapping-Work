## 🗺️ Map Projects

### NDVI Analysis — Lab 6
A Normalized Difference Vegetation Index (NDVI) workflow using
multispectral satellite imagery. Covers band math, raster
reclassification, and the interpretation of vegetation health across a
study area.

📄 [Open PDF](docs/pdfs/NDVI-Lab6-Decneut.pdf)

### Isarithmic Mapping — Geoviz Practice 5
An isarithmic (continuous-surface) map produced for a geovisualization
practice exercise. Explores interpolation methods, classed contour
rendering, and the cartographic conventions that make continuous data
legible.

📄 [Open PDF](docs/pdfs/GeovizPractice5-Isarithmic-Decneut.pdf)
